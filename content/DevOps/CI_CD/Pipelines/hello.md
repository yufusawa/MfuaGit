## Первый Pipeline на CI в GitHub Actions

**Цель** — просто заставить систему **GitHub Actions** работать!

### 1. Создайте на **GitHub** новый публичный репозиторий `my-first-cicd` с `README.md`

В корне репозитория создайте папку `.github/workflows/.`

Внутри создайте файл `hello.yml`. **GitHub** сам предложит вам шаблоны, но для начала скопируйте вот этот простой код, который просто выводит приветствие при каждом пуше.

### 2. Файл `hello.yml`
```yml
name: Мой первый workflow
on: [push] # Запускать при каждом пуше
jobs:
  build:
    runs-on: ubuntu-latest # Использовать последнюю версию Ubuntu
    steps:
      - uses: actions/checkout@v4 # Шаг 1: скачать код из репозитория
      - name: Запустить команду
        run: echo "Привет, мир! Я только что запустил CI!"
```

### 3. Проверить онлайн

- Закоммитьте и запушите в строго в ветку `main` этот файл в ваш репозиторий
- Перейдите на вкладку **Actions** в вашем репозитории на **GitHub**. Вы увидите, как ваш **Workflow** запустился, а через минуту загорится **зеленая** галочка, которая означает, что все шаги прошли успешно
