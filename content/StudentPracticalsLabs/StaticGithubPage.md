# 📝 Самостоятельная работа и Техническое задание: Разработка сайта-визитки с использованием Git и GitHub Pages

## 🎯 Цель проекта

Разработать статический сайт на чистом **HTML**, **CSS** и **JavaScript** с организацией кода в репозитории **Git** и последующим размещением на **GitHub Pages**.

> Рекомендуется использовать нейросеть для генерации кода веб-страниц и README.md

---

## 📋 Основные требования

### 🏗️ Технический стек
- **HTML5** - семантическая разметка
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript** - интерактивность (допускается использование localStorage)
- **Git** - контроль версий
- **GitHub Pages** - хостинг сайта
- **Visual Studio Code**

### 🌐 Примерная структура сайта (3-4 страницы)

1. **index.html** - Главная страница
1. **about.html** - О проекте
1. **contact.html** - Контакты
1. **blog.html** - Блог/Новости

## 🔧 Git workflow (рабочий процесс)

### Ветка `main`:
- ✅ Стабильная версия проекта
- ✅ Только проверенный код
- ✅ Готовая к деплою версия

### Ветка `test`:
- ✅ Экспериментальные функции
- ✅ Новая функциональность
- ✅ Тестовые стили и скрипты

### Процесс работы:
```bash
# 1. Начинаем работу в test ветке
git checkout test

# 2. Разрабатываем новую функцию
# 3. Коммитим изменения
git add .
git commit -m "feat: add dark theme functionality"

# 4. Переключаемся в main
git checkout main

# 5. Мёржим изменения из test
git merge test

# 6. Пушим в удаленный репозиторий
git push origin main
```
